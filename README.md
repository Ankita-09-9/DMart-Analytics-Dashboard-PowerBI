# DMart Sales & Customer Analytics Dashboard

## Project Overview
This Power BI project contains 3 interactive dashboards designed to analyze DMart sales, product performance, and customer behavior using advanced visualizations and business intelligence techniques.

The dashboards include dynamic dark/light mode themes with interactive toggle buttons for enhanced user experience.

## Dashboards Included

### 1. DMart Sales Dashboard
- Monthly Sales & MOM Growth
- Sales by Category
- Sales by Marketing Channel
- Sales by Ship Mode
- Regional Sales Analysis
- Product Sales Table

### 2. Product Analysis Dashboard
- Top & Bottom Products by Sales
- Rating vs Sales Analysis
- Discount vs Sales Analysis
- Category-wise Product Performance
- Product KPI Metrics

### 3. Customer Analytics Dashboard
- Customer Order Trends
- Orders by Age Group & Gender
- Payment Status Analysis
- Market Segment Analysis
- Preferred Payment Methods

## Features
- Interactive multi-page Power BI dashboards
- Navigation buttons for dashboard switching
- Dynamic Dark & Light Theme Toggle
- KPI Cards & Advanced Visualizations
- Interactive Filters & Slicers
- Business Intelligence Reporting
- User-Friendly Dashboard Interface

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Modeling
- Excel

## Key Insights
- Instagram generated the highest sales contribution.
- Local category products dominated sales performance.
- Debit card was the most preferred payment method.
- High-rated products showed better sales trends.
